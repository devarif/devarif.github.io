Tentang

Mengubah Ide Menjadi Produk Digital.

Halo, saya Ahmad Hasan Al Arif Billah — biasa dipanggil Arif. Seorang Frontend & Mobile Developer dengan pengalaman lebih dari 11 tahun di industri teknologi, berbasis di Jakarta dan Surabaya.

Spesialisasi saya mencakup pengembangan web dan aplikasi mobile (Android & iOS) menggunakan React.js, Next.js, React Native, TypeScript, dan JavaScript modern. Saya percaya bahwa produk digital yang baik lahir dari kode yang bersih, arsitektur yang terstruktur, dan pengalaman pengguna yang intuitif.

Di sisi backend, saya memiliki pengalaman solid dengan PHP (Laravel, Lumen, CodeIgniter), Golang, dan Node.js (Express.js) — membangun REST API, microservices, hingga integrasi sistem. Pemahaman fullstack ini memungkinkan saya berkolaborasi secara efektif lintas tim dan memahami siklus pengembangan secara menyeluruh.

Saya selalu bersemangat mempelajari teknologi baru, menyelesaikan tantangan kompleks, dan bekerja sama dengan tim yang memiliki visi untuk menciptakan solusi digital yang berdampak nyata.
